# 🧮 Calculadora em C# Console

Este projeto é uma calculadora desenvolvida em C Sharp para execução no terminal. O programa apresenta um menu interativo, permitindo ao usuário escolher operações matemáticas e navegar entre os menus ou encerrar a aplicação.

---

## 📌 Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* √ Raiz Quadrada
* ^ Exponenciação
* 🔄 Retorno ao menu principal
* 🚪 Encerramento do programa com `exit`

---

## 🛠️ Tecnologias Utilizadas

* C Sharp
* .NET
* Visual Studio Code
