# 📘 Estudos de C#

Bem-vindo ao meu repositório de estudos em **C#**! 🚀

Este espaço foi criado para organizar meus aprendizados, exercícios e projetos desenvolvidos ao longo da minha jornada com a linguagem C# e a plataforma .NET.

---

## 🎯 Objetivo

O objetivo deste repositório é:

* Aprender os fundamentos da linguagem C#;
* Praticar lógica de programação;
* Desenvolver aplicações de console e projetos práticos;
* Estudar programação orientada a objetos (POO);
* Explorar recursos do ecossistema .NET.

---

## 📚 Conteúdo dos Estudos

### 🟢 Fundamentos

* Variáveis e tipos de dados
* Operadores aritméticos e lógicos
* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`, `foreach`)
* Métodos e parâmetros

### 🔵 Programação Orientada a Objetos (POO)

* Classes e objetos
* Encapsulamento
* Herança
* Polimorfismo
* Abstração
* Interfaces

### 🟣 Coleções e Manipulação de Dados

* Arrays
* Listas (`List<T>`)
* Dicionários (`Dictionary<TKey, TValue>`)
* LINQ

### 🟠 Tratamento de Exceções

* `try`, `catch`, `finally`
* Criação de exceções personalizadas

### 🔴 Projetos Práticos

* Calculadora
* Sistema de cadastro
* Gerenciador de tarefas
* Jogo simples no console

## ▶️ Como Executar os Projetos

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/estudos-csharp.git
```

2. Acesse a pasta do projeto:

```bash
cd estudos-csharp
```

3. Execute o projeto desejado:

```bash
dotnet run
```
